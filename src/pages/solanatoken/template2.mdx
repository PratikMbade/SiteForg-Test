### 1.Navigation Links

You can create a navigation bar with links to your social media and other platforms using the `NavigationBar` component. This makes it easy for users to access your channels and resources.

#### Example Code

First, define an array of links for your social platforms:

```tsx
const linksOfSocails = [
  { telegramLink: "https://t.me/your-telegram-channel" },
  { xLink: "https://twitter.com/your-handle" },
  { dexScreener: "https://dexscreener.com" },
  { jupiter: "https://jup.ag" },
];
```

```tsx
<NavigationBar linksOfSocails={linksOfSocails} />
```

### 5. Feature section and Buy button

```tsx
import { Template4FeatureSection } from "@/components/template-components/template4-featuresection";
```

### Template4FeatureSection Component

The `Template4FeatureSection` component allows you to display an image along with a button. You can customize the image source, button text, and the image's width and height by passing props to the component.

#### Usage Example:

```tsx
<Template4FeatureSection
  ImgUrlOrPath="/Alien02.png"
  TextOnButton="Buy SolAlienz"
  ImgWidth={500}
  ImgHeight={500}
/>
```
### 6.Tokenomics Section

The Tokenomics section displays important information about your token, such as its ticker symbol, total supply, and tax rate. You can customize this section using the `Template4Tokennomics` component by providing the necessary props.

#### Example Code

Here is an example of how to use the `Template4Tokennomics` component:

```tsx
<Template4Tokennomics
  tokenTicker={"$SolAlienz"}
  tokenSupply={"1 Billion"}
  tokenTax={"0%"}
  imageUrl={"/Alien03.png"}
/>
