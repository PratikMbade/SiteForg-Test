### 1.Navigation Links

```tsx
import { Template1Navbar } from "@/components/template-components/template1-navbar";
```

You can create a navigation bar with links to your social media and other platforms using the `NavigationBar` component. This makes it easy for users to access your channels and resources.

#### Example Code

First, define an array of links for your social platforms:

```tsx
const linksOfSocails = [
  { telegramLink: "https://t.me/your-telegram-channel" },
  { xLink: "https://twitter.com/your-handle" },
  { dexScreener: "https://dexscreener.com" },
  { jupiter: "https://jup.ag" },
];
```

```tsx
<Template1Navbar linksOfSocails={linksOfSocails} />
```

### 6. Tokenomics

```tsx
import { Template1Tokenomics } from "@/components/template-components/template1-tokenomic";
```

### Tokenomics Component

The Tokenomics section displays important information about your token, such as its ticker symbol, total supply, and tax rate. You can customize this section using the `Template4Tokennomics` component by providing the necessary props.

#### Example Code

```tsx
<Template1Tokenomics
  tokenTicker={"BlobberTicker"}
  tokenSupply={"1 Billion"}
  tokenTax={"0% Tax"}
/>
```
